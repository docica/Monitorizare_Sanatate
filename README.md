Monitorizare-Sanatate

Aplicatia este conceputa pentru monitorizarea persoanelor cu probleme cardiace si diabetice. Aplicatia permite operatii de adaugare, editare si stergere pentru datele pacientului si masuratorile cardiovasculare si ale glicemiei. Functionalitati principale: monitorizarea glicemiei, monitorizarea tensiunii, sisteme de alerta (generarea unor mesaje de avertizarea in momentul in care valorile glicemiei sau ale tensiunii pun in pericol viata pacientului), istoric medical.
